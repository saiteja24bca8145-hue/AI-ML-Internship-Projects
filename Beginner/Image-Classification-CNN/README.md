# Image Classification using CNN

## Objective
Develop an image classification system using Deep Learning to classify images into different categories.

## Dataset
CIFAR-10 Dataset

## Model Used
Convolutional Neural Network (CNN)

## Classes
- Airplane
- Automobile
- Bird
- Cat
- Deer
- Dog
- Frog
- Horse
- Ship
- Truck

## Results
Training Accuracy: 77.15%

Validation Accuracy: 68.19%

## Libraries Used
- TensorFlow
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Conclusion
A CNN model was successfully trained on the CIFAR-10 dataset and achieved good image classification performance across 10 classes.
